\doxysubsection{Detailed Description}
A utility class for handling file operations related to encoded and decoded data. 

The \doxylink{class_file_handler}{File\+Handler} class provides methods to read encoded data from a file and write decoded data back to a file. It supports operations with 2D vectors of integers representing the data.
