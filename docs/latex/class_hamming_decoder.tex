\doxysubsection{Detailed Description}
A class for decoding data encoded with Hamming codes. 

The \doxylink{class_hamming_decoder}{Hamming\+Decoder} class provides methods to decode data, check for errors, and calculate syndromes to identify and correct single-\/bit errors. It utilizes a parity-\/check matrix for these operations.
