\doxysubsection{Detailed Description}
A class for encoding data using the Hamming code technique. 

The \doxylink{class_hamming_encoder}{Hamming\+Encoder} class provides methods to encode data using a predefined generator matrix for error detection and correction. The class initializes the generator matrix automatically upon creation.
