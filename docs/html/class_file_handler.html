Detailed Description

A utility class for handling file operations related to encoded and decoded data.

The FileHandler class provides methods to read encoded data from a file and write decoded data back to a file. It supports operations with 2D vectors of integers representing the data.

std::vector< std::vector< int > > FileHandler::readEncodedFile ( const std::string & filename)

Reads encoded data from a file.

This function reads a file containing encoded data, parses it, and returns the data as a 2D vector of integers. Each row in the file corresponds to a vector of encoded bits.

Parameters

Returns
A 2D vector of integers representing the encoded data.

This method reads a file containing lines of 7-bit encoded words. Each line in the file must contain exactly 7 bits (0s and 1s). Lines that do not meet this criterion are skipped with an error message.
