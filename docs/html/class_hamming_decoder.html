Detailed Description

A class for decoding data encoded with Hamming codes.

The HammingDecoder class provides methods to decode data, check for errors, and calculate syndromes to identify and correct single-bit errors. It utilizes a parity-check matrix for these operations.

Calculates the syndrome for the given encoded data.

Calculates the syndrome for a given encoded data vector.

The syndrome is calculated using the parity-check matrix and provides information about error positions in the encoded data.

Parameters
encodedDataA vector of integers representing the encoded data bits (0s and 1s).
Returns
A vector of integers representing the calculated syndrome.

This function computes the syndrome by performing a modulo-2 dot product of the parity-check matrix (H^T) and the encoded data vector.

Parameters

A 3-bit vector representing the syndrome.
